## 文档规范

### Markdown格式
- 使用标准Markdown语法
- 代码块使用语法高亮
- 图片放在`images/`目录
- 使用相对路径引用图片

### 必需内容
每个文档应包含：
1. 标题和概述
2. 目录（长文档）
3. 详细内容
4. 参考链接（如有）
5. 更新记录（可选）
